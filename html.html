Using the Node.js Solver (cap.js)

The cap.js solver produces the same frc-captcha-solution token as the browser widget, so you can automate form submissions server-side: 

// Fetch a puzzle and solve it in one call:
const { fetchAndSolvePuzzle } = require('./cap');

const token = await fetchAndSolvePuzzle('FCMSPLFFSPQ6TH80');
// token = "f7b1…833.aanq….AAAAAE8mAAA=.AgAA"

// Submit to your target URL with the token:
await fetch('https://example.com/login', {
  method: 'POST',
  body: new URLSearchParams({
    username: 'alice',
    password: 'secret',
    'frc-captcha-solution': token,
  }),
});

// Or solve an already-fetched puzzle string:
const { solvePuzzleString } = require('./cap');
const token2 = await solvePuzzleString('REPLACE_WITH_ACTUAL_PUZZLE_STRING');

Token format

The frc-captcha-solution token is a dot-separated string with four parts: 

<signature>.<puzzleBase64>.<solutionBase64>.<diagnosticsBase64>

signature       — hex string identifying the account key
puzzleBase64    — the base64 puzzle data from the API response
solutionBase64  — base64( 8 bytes × numPuzzles ) nonces that satisfy hash < threshold
diagnosticsBase64 — base64( [solverID, timeHigh, timeLow] ) — 3 bytes
